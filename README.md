# Submission-search

- Language : C++ 10.0.5
- OS : macOS BigSur
- IDE : Xcode
- Library : siv3d 0.4.3

Siv3dはXcodeProjectなので、Xcodeを使用してください。一部コメントアウトしていますが、使用している関数の説明や復元アルゴリズムの手順等は省略。
